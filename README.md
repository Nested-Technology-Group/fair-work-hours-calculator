## What is NestedClock?

[NestedClock](https://nestedclock.com.au) is a cloud-based time & attendance system built from the ground up for Australian businesses. It replaces paper timesheets, spreadsheets, and guesswork with one modern platform.

### Features

| Category | Highlights |
|----------|-----------|
| Clock-in & Verification | PIN auth, photo capture, GPS stamps, kiosk mode |
| Scheduling & Monitoring | Roster builder, real-time dashboard, multi-location |
| Payroll & Exports | Xero/MYOB exports, PAYG estimation, 85+ AU holidays |
| Communication | Staff portal, messaging, iOS & Android apps |
| Configuration | Multi-timezone, custom week start, late alerts |
| AI-Powered | Admin assistant (31 tools), agentic AI (19 actions) |

### Key Highlights

- 🇦🇺 100% Australian data residency
- 🔒 Tenant isolation — each business gets its own database
- 📱 Native iOS & Android apps
- 🤖 AI powered by Claude Sonnet 4.6
- 💰 Free plan available, 14-day trial on all paid plans
